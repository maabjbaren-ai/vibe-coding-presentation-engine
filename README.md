## 1. תיאור הפרויקט ומטרות
פרויקט זה מיישם מטא-מצגת אינטראקטיבית בת 20 שקופיות המבוססת על הארכיטקטורה הפדגוגית של הספר "ארכיטקטורת מולטימדיה מונחית Vibe Coding". המערכת מפרידה לחלוטין בין כוונת התוכן (המוגדרת בקובץ JSON מובנה) לבין מנוע הרינדור הוויזואלי (סקריפט Python המייצר דף HTML מבוסס Reveal.js).

## 2. ארכיטקטורת תמיכה ב-RTL ומניעת "קוד שבור"
כדי לעמוד בדרישות הקורס ולהימנע מרשימות שבורות ופיסוק קופץ, הפרויקט מיישם פתרונות CSS ארכיטקטוניים קשיחים:
* **ביטול תגיות רשימה סטנדרטיות:** המצגת משתמשת במבנה Flexbox מותאם אישית (`display: flex` ו-`align-items: flex-start`) ליישור ועיצוב תבליטים (Bullets) מצד ימין בצורה חסינה.
* **בקרת דו-כיווניות (BiDi):** החלת המאפיין `unicode-bidi: embed` על תיבות הטקסט כדי למנוע היפוך של מספרים, ראשי תיבות ומילים לועזיות (כמו AI, JSON, Flexbox) בתוך משפטים בעברית.
